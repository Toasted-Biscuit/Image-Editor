# Image Editor
A basic image editor that can add filters and do a few effects

## Current filters and effects:
+ Flip Horizontally
+ Flip Vertically
+ Grayscale
+ Blur
+ Contrast
+ Posterize
+ Scramble
+ Saving the Image

## Controls
+ Reset Image = **r**
+ Flip Horizontally = **h**
+ Flip Vertically = **v**
+ GrayScale = **g**
+ Blur = **b**
+ Contrast = **c**
+ Posterize = **p**
+ Scramble = **x**
+ Save Image = **s**
