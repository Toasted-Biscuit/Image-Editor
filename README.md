# Image Editor
A basic image editor that can add filters and do a few effects

## Current filters and effects:
+ Flip Horizontally
+ Flip Vertically
+ Grayscale
+ Blur
+ Scramble
+ Contrast
+ Posterize
+ Saving the Image

## Controls
+ Reset Image = **r**
+ Flip Horizontally = **h**
+ Flip Vertically = **v**
+ GrayScale = **g**
+ Blur = **b**
+ Scramble = **x**
+ Contrast = **c**
+ Posterize = **p**
+ Save Image = **s**
